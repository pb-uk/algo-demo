## Development

1. Install Gridsome CLI tool
   ```bash
   npm i -g @gridsome/cli
   ```

2. Start a local dev server at http://localhost:8080/
   ```bash
   gridsome develop
   ```

3. Create static files for deployment in `./dist`.
   ```bash
   gridsome build
   ```
